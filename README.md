# Easy Ender

Easy Ender is a Fabric mod for Minecraft 1.20.1 that adds new ways to obtain Ender Pearls through mining, processing, and risky consumption.

Instead of relying on Endermen or trading, players can now mine Ender Ore, refine it into fragments, and craft or process their way to pearls.

All variants:

- Support **Silk Touch**
- Emit a faint glow
- Drop **Ender Fragments**
