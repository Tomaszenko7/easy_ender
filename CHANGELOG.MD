# Easy Ender – Changelog

## v1.0.0 – Initial Release

### Added
- Ender Ore (Overworld)
- Deepslate Ender Ore
- End Ender Ore (The End)
- Ender Fragments
- Ender Dust
- Ender Pearl crafting via fragments
- Smelting & Blasting support
- Create Crushing & Milling support
- XP drops from ores
- Silk Touch support
- Custom ore textures
- Custom advancements:
    - Fragmented Reality
    - Spontaneous Relocation
- Eating Ender Fragments causes teleportation
- Enderman sound & particle effects
- Cooldown, nausea & damage balancing
- End-weighted ore generation

### Balance
- Overworld ores are rare
- End ores are common
- Ore blocks always smelt to 1 fragment
- Crushing allows ore doubling

### Technical
- Fully data-driven worldgen
- Fabric 1.20.1 compatible
- Server & client safe
